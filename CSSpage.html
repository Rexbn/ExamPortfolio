CSS Syntaks

Vælg hvad der skal styles (Selectors)

- Alle tags af en bestemt type (p, h1, h2, body, a:link..)
- flere udvalgte elementer (.class)
- et helt specifikt element (#id)

En række deklarationer (declarations)

- Deklarationen startes med { (krøllet parantes) og slutes med den tilsvarende }
- består af egenskaber (properties) og værdier (values) adskilt af : (kolon)
- Egenskaber afsluttes med ; (semikolon)

Kaskaderegler

Nedarvning (Inheritance)
Cascading

Et sæt regler der definerer hvordan modstridende CSS deklareringer løses.
- Specificity: Mere specifik har forrang mod mere generel
- Source order: Seneste deklareringer har forrang mod tidligere deklaringer
- Specificity har forrang mod source order
- Og undtagelsen: !important - men den smadrer totalt ideen med kaskaderegler - undgå den så vidt muligt.
