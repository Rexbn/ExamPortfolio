Text Styles

Hvad er fonts?

Fonts er et set af symboler. Det er udseendet på texten.

Der findes forskellige font families, såsom cursive(håndskrivning), fantasy(Dekorativ), Monospace(samme brede), Sans-Serif (Text uden fødder) og Serif (text med fødder).

Anskaffelse af fonts

Web-Safe Fonts

fonts der er supported af flere browsers.

Web Fonts

Browser downloaded fonts fra f.eks. Google-fonts.
Brug ikke for mange downloaded fonts. Det gør websiden langsommere.

Costum Fonts

Download til egen pc. - Støttet af største delen af webbrowsers.
Billedet til venstre viser, at når man har valgt en font fra google.fonts, skal man blot kopiere disse links. Dermed hoster google fontsne.

Styles

Fonts kan have specifikke styles, såsom Normal(Default), Italic(Skråskrift) og Oblique (lidt mere skråskrift).

Størrelser

Fonts kan også have forskellige størrelser der bestemmes således:

  • Px (pixel)
  • x-small eller xx-small (Ekstra small)
  • em (nuværende font størrelse)
  • rem (root element)
  • vh (1% af viewport højden)

Tykkelse og farver

Derudover kan fonts også have en "vægt".
Den defineres sådan: Normal (default), bold, lighter eller bolder.
Det gør fonten enten tykkere eller tynder.

Farver

Fonts kan også farves med lige netop den farve man ønsker.
Det skal skrives ind som en hex color kode.
Hvis man ønsker det, kan man også putte opacity på farven, som går fra 0 til 1, hvor 1 er slet ikke gennemsigtig.

Kerning

Fjerner rummene mellem bogstaver.
Auto(default) - Browseren vurdere selv om der skal bruges kerning eller ikke.
Normal (Forced on)
None (Forced off)

Stretch

  • Ultra condensed
  • Extra condensed
  • Condensed
  • Semi condensed
  • Normal
  • Semi Expanded
  • Expanded
  • Extra Expanded
  • Ultra expanded

Shadows

Med text-shadow, kan man få skygge på sin tekst. Den har en blur med default 0. Når man bruger text-shadow, kan det skrives sådan: "text-shadow: 0px 2px 3px #fff;". Den første pixel er X-offset. De 2px er Y-offset. 3px for blur og #fff for hvilken farve blurren skal have.

Capitalization

I stedet for at direkte skrive alt i f.eks. capslock, kan man skrive "text-transform:uppercase". Derefter vil alt man skriver indenfor denne regl, være capslock.
